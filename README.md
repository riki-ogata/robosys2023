# robosys2023
![test](https://github.com/riki-ogata/robosys2023/actions/workflows/test.yml/badge.svg)

本リポジトリは, ロボットシステム学2023において作成したものである.

## インストール方法
```
$ git clone git@github.com:riki-ogata/robosys2023.git
$ cd robosys2023
$ ls
LICENSE  README.md  plus  test.bash
```
## plusコマンド
```
$ seq 5 | ./plus
15
```
